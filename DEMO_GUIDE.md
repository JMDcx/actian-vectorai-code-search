# Actian VectorAI DB 黑客松演示指南

## 🎯 视觉效果特点

### 赛博朋克科技风格
- **霓虹色彩系统**：使用高饱和度的霓虹绿、粉、蓝色
- **动态背景**：网格背景 + 扫描线动画 + 漂浮粒子
- **发光效果**：所有节点和连接都有发光滤镜效果
- **数据流动画**：连接线有流动效果，展示数据传输
- **脉冲动画**：重要元素有呼吸灯效果

### 代码语义聚类可视化
- **形状区分**：
  - 类 → 六边形
  - 函数 → 圆形
  - 导入 → 菱形
- **大小变化**：节点大小根据代码复杂度动态调整
- **颜色编码**：
  - Python → 霓虹绿 (#00ff88)
  - JavaScript → 霓虹粉 (#ff00ff)
  - TypeScript → 霓虹蓝 (#00ffff)
- **语义力布局**：相似功能的代码自动聚在一起

### 代码复杂度分析
- **4级复杂度分级**：
  - 简单 (<10行) → 绿色
  - 中等 (10-30行) → 黄色
  - 复杂 (30-50行) → 橙色
  - 非常复杂 (>50行) → 红色
- **复杂度指示器**：高复杂度节点有中心白点
- **进度条动画**：复杂度条有闪光效果

### 实时统计仪表盘
- **向量数据库统计**：
  - 代码片段总数
  - 语义连接数
  - 平均相似度
- **语义聚类分析**：按语言和类型分组统计
- **复杂度分布**：可视化复杂度分布情况

## 🎨 演示技巧

### 视觉冲击点
1. **开场** - 让系统运行 5 秒钟，展示扫描线和粒子动画
2. **交互** - 快速拖拽几个节点，展示力场布局的动态调整
3. **悬停** - 在不同类型的节点上悬停，展示高亮效果
4. **点击** - 选择一个高复杂度的节点，展示详细分析

### 技术亮点强调
- **相似度 98%+** - 展示 VectorAI DB 的精确度
- **毫秒级响应** - 快速点击多个节点，展示实时性能
- **语义理解** - 搜索"database"、"api"等功能关键词
- **本地部署** - 提及无需云服务，完全离线运行

### 故障准备
- **数据为空** - 准备一个演示项目预索引
- **性能问题** - 降低节点数量到 50 个
- **浏览器兼容** - 建议使用 Chrome 最新版本
